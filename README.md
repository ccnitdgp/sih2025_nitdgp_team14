- EASY OF ACCESS FOR EACH GROUP OF LITERACY.
- - ICON BASED NAVIGATION, MULTILINGUAL VOICE ASSISTANCE, MULTILINGUAL INTERFACE AND MULTILINGUAL CHATBOT SUPPORT.

- FOLLOWS ALL 5 SDGs SUGGESTED BY WORLD HEALTH ORGANISATION AND UNITED NATIONS UNDER HEALTHTECH/HEALTHCARE.
